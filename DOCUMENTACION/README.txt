La documentación html es documentación interna, no para entregar. Escribiremos el la documentación interna aspectos relacionados con el desarrollo del proyecto con respecto a cómo hacemos las cosas, explicaciones, los actas de las reuniones.
La documentación .doc será la que se formatee para entregar.

Para añadirlas al menú simplemente añadir el fichero a la carpeta Html y un enlace en el orden adecuado en menu.html.
La plantilla para las entradas es la siguiente:
<!DOCTYPE html>
<html>
 <head>
	<meta http-equiv="Content-Type" content="text/html">
	<title>Presentación</title>
	<meta http-equiv="Content-Language" content="es">
	<meta charset='utf-8'>
	<link rel="stylesheet" type="text/css" href="style1.css">
 </head>
<body>

</body>
</html>
